项目经历 (Projects)

GPT-Poet：基于深度 Transformer 的古诗生成系统

基于 PyTorch 开发的自动化诗歌生成系统，通过对近 15 万首高量古诗词语料的深度建模，实现高文学性与严格格律的文本生成。

  • 自回归架构： 采用深度 Transformer Decoder 架构，配合 下三角因果掩码 (Causal Mask) 阻断前向信息泄露，实现标准的从左至右逐字生成逻辑。
  • 生成策略优化： 自主实现带温度控制的 Top-P 核采样 (Nucleus Sampling) 与重复惩罚机制，有效解决了传统模型的“复读机”问题，提升了文本多样性。
  • 控制与可解释性： 通过序列掩码实现精准的诗词格式控制，并利用预测概率热力图对模型内部决策逻辑进行可视化分析。
PyTorch Transformer NLP Generative AI

Mini-Kernel：用户态微内核仿真系统

基于 C/S 架构，在用户态模拟操作系统核心机制，重点探索存储优化与并发控制。

  • 架构解耦： 设计了基于共享内存的异步消息队列，成功解耦 GUI 与内核进程。
  • 存储引擎优化： 为自研 FAT 文件系统设计了采用 Write-Back 策略的 LRU 缓冲池，将 I/O 次数降低了约 40%。
  • 并发控制： 引入读者-写者锁（RW-Lock）机制，解决多用户并发访问冲突，提升系统吞吐量。
C/C++ Operating System LRU Buffer Pool Concurrency

MIPS-Lite：五级流水线处理器核

从 RTL 级出发，完整实现支持 56 条 MIPS32 指令的五级流水线 CPU 核。

  • 流水线冒险控制： 摒弃简单的全局停顿，独立设计动态数据前递（Forwarding）与 Load-Use 互锁机制，将吞吐率提升约 18%。
  • 自动化验证： 构建基于指令轨迹比对的验证框架，系统验证极端冒险场景下的硬件行为。
Verilog Computer Architecture CPU Design
